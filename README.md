# Go Todo API

### Available Endpoints:

Get all todos (`GET`) : 
```
/todos
```
Get a specific todo list (`GET`):
```
/todolist/:id
```
Get a specific todo (`GET`):
```
/todo/:id
```
Update a specific todo (`PUT`):
```
/todo/:id
```
Delete a specific todo (`DELETE`):
```
/todo/:id
```
Create a new todo list (`POST`):
```
/todolist
```
